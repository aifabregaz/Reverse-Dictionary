Reverse Dictionary provides creation of reverse dictionaries from different text sources.

Third party dependencies:
    -LemmaGen - Lemmatisation Open Source Framework <http://lemmatise.ijs.si/>
    -Apache NPOI - using for reading docx files without dependencies from MS Office <http://code.google.com/p/npoi/>
    -NetOffice - .NET Wrapper Assemblies for accessing MS Office applications (used for reading doc files) <http://netoffice.codeplex.com/>
